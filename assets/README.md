# Assets

This directory contains the PNG assets for the application. Expo expects PNG files at fixed paths.

- `icon.png`: 1024×1024 application icon
- `adaptive-icon.png`: 1024×1024 adaptive icon (Android)
- `splash.png`: Splash screen 1242×2688
- `favicon.png`: Web favicon 48×48

The paths above match `app.json`.
